# Debug Issue

Systematically investigate and resolve the described bug or error.

## Instructions
1. Understand the error/unexpected behavior
2. Identify relevant files and code paths
3. Form hypotheses about the cause
4. Suggest diagnostic steps
5. Propose a fix

## Process
- Start with the error message/symptoms
- Trace the code path
- Check common issues for this stack (Next.js, Drizzle, Supabase)
- Consider recent changes

## Output Format
- Observed vs expected behavior
- Root cause analysis
- Specific fix with code
- Verification steps
