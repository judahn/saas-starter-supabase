# Plan Task

Create a detailed implementation plan for the requested task.

## Instructions
1. Read the project CLAUDE.md for context
2. Understand the current state of relevant files
3. Break down the task into specific steps
4. Identify risks and dependencies
5. Provide a clear execution order

## Output Format
- Summary of the task
- Prerequisites and assumptions  
- Numbered implementation steps
- Files that will be modified
- Testing/verification plan
- Potential issues to watch for

Use the planner agent mindset: think strategically before coding.
